# 🚀 การตั้งค่าระบบ PromptPay QR Code (ฉบับย่อ)

## ⚡ เริ่มใช้งานใน 3 ขั้นตอน

### 1. ตรวจสอบว่าคุณมี PromptPay แล้ว
- เปิดแอพธนาคาร → เมนู "PromptPay" หรือ "พร้อมเพย์"
- ถ้ายังไม่มี: ลงทะเบียนผูกกับเบอร์โทรศัพท์หรือเลขบัตรประชาชน

### 2. แก้ไขไฟล์ `freezefood/src/app/checkout/checkout.component.ts`
```typescript
// บรรทัดที่ 24-25
shopPromptPayId: string = '0891234567'; // ⬅️ เปลี่ยนเป็นเบอร์โทรของคุณ (10 หลัก)
shopName: string = 'ร้านของฉัน'; // ⬅️ เปลี่ยนเป็นชื่อร้านของคุณ
```

### 3. รีสตาร์ทเซิร์ฟเวอร์
```bash
# กด Ctrl+C เพื่อหยุด
# จากนั้นรันใหม่
cd freezefood
npm start
```

4. 📱 คุณได้รับการแจ้งเตือนจากธนาคาร

## 💡 ตัวอย่างการใช้งาน

### ใช้เบอร์โทร (แนะนำ):
```typescript
shopPromptPayId: string = '0812345678'; // 10 หลัก ขึ้นต้นด้วย 0
```

### ใช้เลขบัตรประชาชน:
```typescript
shopPromptPayId: string = '1234567890123'; // 13 หลัก
```

---

## ⚠️ หมายเหตุสำคัญ

1. **ไม่มีค่าใช้จ่าย** - ใช้ PromptPay ฟรี ไม่มีค่าธรรมเนียม
2. **เงินเข้าทันที** - ไม่ต้องรอ เงินเข้าบัญชีทันทีที่ลูกค้าโอน
3. **รองรับทุกธนาคาร** - ใช้ได้กับทุกธนาคารในไทย
4. **ปลอดภัย** - ใช้มาตรฐาน PromptPay ของ ธปท.

---

## ❓ มีปัญหา?

### QR Code ไม่แสดง?
- ตรวจสอบเบอร์โทรว่าถูกต้อง (10 หลัก)
- ตรวจสอบว่ารีสตาร์ทเซิร์ฟเวอร์แล้ว
- เปิด Console (F12) ดู error

### แสกนแล้วไม่สามารถโอนได้?
- ตรวจสอบว่าเบอร์ผูกกับ PromptPay แล้ว
- ลองใช้แอพธนาคารอื่น
