## 🎯 ภาพรวม

ระบบ QR Code PromptPay ช่วยให้ลูกค้าสามารถชำระเงินผ่านแอพธนาคารโดยตรง โดยไม่มีค่าใช้จ่ายเพิ่มเติม เงินจะเข้าบัญชีของร้านค้าทันที

### ✨ คุณสมบัติหลัก:
- ✅ **ฟรี 100%** - ไม่มีค่าธรรมเนียมใดๆ
- ✅ **รวมยอดเงินอัตโนมัติ** - QR Code มียอดเงินที่ต้องชำระแล้ว
- ✅ **รองรับทุกธนาคาร** - ใช้ได้กับแอพธนาคารทุกธนาคารในไทย
- ✅ **เงินเข้าทันที** - เงินเข้าบัญชีร้านค้าทันทีที่ลูกค้าโอน
- ✅ **ปลอดภัย** - ใช้ระบบ PromptPay มาตรฐานของธนาคารแห่งประเทศไทย

## ⚙️ วิธีการตั้งค่า

### สำหรับเจ้าของร้าน

#### 1. เตรียมข้อมูล PromptPay
คุณต้องมีเบอร์โทรศัพท์หรือเลขบัตรประชาชนที่ผูกกับ PromptPay แล้ว

**วิธีตรวจสอบว่าคุณมี PromptPay หรือยัง:**
1. เปิดแอพธนาคารของคุณ
2. ไปที่เมนู "PromptPay" หรือ "พร้อมเพย์"
3. ตรวจสอบว่าเบอร์โทรศัพท์หรือเลขบัตรประชาชนของคุณผูกกับบัญชีแล้วหรือยัง

**ถ้ายังไม่มี PromptPay:**
1. เปิดแอพธนาคาร
2. เลือกเมนู "PromptPay" หรือ "พร้อมเพย์"
3. เลือก "ลงทะเบียน"
4. เลือกผูกด้วย "เบอร์โทรศัพท์" (แนะนำ) หรือ "เลขบัตรประชาชน"
5. ยืนยันการลงทะเบียน

#### 2. ตั้งค่าในระบบ

เปิดไฟล์: `freezefood/src/app/checkout/checkout.component.ts`

หาบรรทัดนี้ (บรรทัดที่ 24-25):
```typescript
shopPromptPayId: string = '0812345678'; // เปลี่ยนเป็นเบอร์โทรศัพท์ของคุณ
shopName: string = 'ร้านอาหารแช่แข็ง'; // ชื่อร้านของคุณ
```

**เปลี่ยนค่าดังนี้:**
```typescript
shopPromptPayId: string = '0891234567'; // ใส่เบอร์โทรศัพท์ของคุณ (10 หลัก)
shopName: string = 'ร้านอาหารแช่แข็งของฉัน'; // ใส่ชื่อร้านของคุณ
```

**ตัวอย่าง:**
- ใช้เบอร์โทร: `'0891234567'`
- ใช้เลขบัตรประชาชน: `'1234567890123'` (13 หลัก)

#### 3. รีสตาร์ทเซิร์ฟเวอร์
หลังจากแก้ไขเสร็จแล้ว ให้รีสตาร์ทเซิร์ฟเวอร์ Angular:
```bash
# กด Ctrl+C เพื่อหยุดเซิร์ฟเวอร์
# จากนั้นรันใหม่
cd freezefood
npm start
```

## 🛒 วิธีใช้งานสำหรับลูกค้า

### ขั้นตอนการชำระเงินด้วย QR Code PromptPay

#### 1. เลือกสินค้าและไปหน้าชำระเงิน
- เลือกสินค้าที่ต้องการใส่ตะกร้า
- กดปุ่ม "ชำระเงิน" เพื่อไปหน้า Checkout

#### 2. กรอกข้อมูลการจัดส่ง
- กรอกชื่อ-นามสกุล
- กรอกเบอร์โทรศัพท์
- กรอกที่อยู่จัดส่ง

#### 3. เลือกวิธีชำระเงิน "สแกน QR Code"
- คลิกเลือก "สแกน QR Code"
- QR Code จะแสดงขึ้นมาพร้อมยอดเงินที่ต้องชำระ

#### 4. สแกน QR Code ด้วยแอพธนาคาร
1. เปิดแอพธนาคารของคุณ (ธนาคารใดก็ได้)
2. เลือกเมนู "สแกน QR" หรือ "QR Payment"
3. สแกน QR Code บนหน้าจอ
4. **ตรวจสอบยอดเงิน** ให้ตรงกับที่หน้าจอแสดง
5. ตรวจสอบชื่อผู้รับเงิน (ควรเป็นชื่อร้าน)
6. กดยืนยันการโอนเงิน

#### 5. แนบหลักฐานการชำระเงิน
1. แคปหน้าจอหรือถ่ายภาพหลักฐานการโอนเงินจากแอพธนาคาร
2. กดปุ่ม "Choose File" หรือ "เลือกไฟล์"
3. เลือกรูปภาพหลักฐานการโอน

#### 6. ยืนยันการสั่งซื้อ
- กดปุ่ม "ยืนยันการสั่งซื้อ"
- รอระบบประมวลผล
- จะได้รับหมายเลขคำสั่งซื้อ

## 👨‍💼 วิธีใช้งานสำหรับเจ้าของร้าน

### การรับเงิน

เมื่อลูกค้าสแกน QR Code และโอนเงิน:
- ✅ เงินจะเข้าบัญชีธนาคารของคุณ**ทันที**
- ✅ คุณจะได้รับการแจ้งเตือนจากแอพธนาคาร
- ✅ สามารถตรวจสอบได้ในประวัติการทำรายการของธนาคาร

### การตรวจสอบคำสั่งซื้อ

1. เข้าหน้า Admin (`/admin`)
2. ดูคำสั่งซื้อที่มีสถานะ "รอดำเนินการ"
3. ตรวจสอบหลักฐานการโอนเงินที่ลูกค้าแนบมา
4. ตรวจสอบในบัญชีธนาคารว่าเงินเข้าจริง
5. อัปเดตสถานะคำสั่งซื้อเป็น "กำลังเตรียม" หรือ "จัดส่งแล้ว"

### การอัปเดตสถานะการชำระเงิน

ในหน้า Admin:
1. คลิกที่คำสั่งซื้อที่ต้องการ
2. ตรวจสอบหลักฐานการโอน
3. เลือกสถานะการชำระเงิน:
   - **"ยังไม่ชำระ"** - ยังไม่ได้รับเงิน
   - **"ชำระแล้ว"** - รับเงินเรียบร้อย
4. กดปุ่ม "อัปเดตสถานะการชำระเงิน"

## ❓ คำถามที่พบบ่อย (FAQ)

### Q1: มีค่าใช้จ่ายไหม?
**A:** ไม่มีเลย! การใช้ PromptPay ไม่มีค่าธรรมเนียมทั้งสำหรับผู้โอนและผู้รับเงิน

### Q2: เงินเข้าบัญชีเมื่อไหร่?
**A:** ทันที! เมื่อลูกค้าโอนเงิน คุณจะได้รับเงินในบัญชีธนาคารทันที

### Q3: ใช้ได้กับธนาคารไหนบ้าง?
**A:** ทุกธนาคารในประเทศไทยที่รองรับ PromptPay เช่น:
- ธนาคารกสิกรไทย (K PLUS)
- ธนาคารไทยพาณิชย์ (SCB Easy)
- ธนาคารกรุงเทพ (Bangkok Bank Mobile Banking)
- ธนาคารกรุงไทย (Krungthai NEXT)
- และธนาคารอื่นๆ ทั้งหมด

### Q4: QR Code มียอดเงินอยู่แล้วหรือเปล่า?
**A:** ใช่! QR Code ที่สร้างขึ้นจะมียอดเงินที่ต้องชำระอยู่แล้ว ลูกค้าไม่ต้องกรอกยอดเอง

### Q5: จะรู้ได้ยังไงว่าลูกค้าโอนเงินจริง?
**A:** 
1. คุณจะได้รับการแจ้งเตือนจากแอพธนาคาร
2. ลูกค้าต้องแนบหลักฐานการโอนมาด้วย
3. ตรวจสอบในแอพธนาคารหรือบัญชีของคุณ

### Q6: ถ้าลูกค้าโอนผิดยอดจะเกิดอะไรขึ้น?
**A:** เนื่องจาก QR Code มีจำนวนเงินอยู่แล้ว แอพธนาคารจะไม่ให้ลูกค้าเปลี่ยนยอดได้ ช่วยลดความผิดพลาด

### Q7: จะเปลี่ยนเบอร์รับเงินยังไง?
**A:** แก้ไขในไฟล์ `checkout.component.ts` บรรทัดที่ 24:
```typescript
shopPromptPayId: string = 'เบอร์ใหม่ของคุณ';
```

### Q8: สามารถใช้เลขบัตรประชาชนแทนเบอร์โทรได้ไหม?
**A:** ได้! แค่ใส่เลขบัตรประชาชน 13 หลักแทน:
```typescript
shopPromptPayId: string = '1234567890123';
```

### Q9: ถ้า QR Code ไม่แสดงผลจะทำยังไง?
**A:** ตรวจสอบว่า:
1. ใส่เบอร์โทรศัพท์หรือเลขบัตรประชาชนถูกต้อง
2. เบอร์โทรต้องเป็น 10 หลัก และขึ้นต้นด้วย 0
3. เลขบัตรประชาชนต้องเป็น 13 หลัก
4. ดู Console (F12) เพื่อดู error message

### Q10: ต้องทำอะไรบ้างก่อนเริ่มใช้งาน?
**A:** 
1. ✅ ลงทะเบียน PromptPay กับธนาคาร
2. ✅ ตั้งค่า `shopPromptPayId` ในโค้ด
3. ✅ ตั้งค่า `shopName` ให้เป็นชื่อร้านของคุณ
4. ✅ รีสตาร์ทเซิร์ฟเวอร์

## 🔧 การแก้ปัญหา

### ปัญหา: QR Code ไม่แสดงผล
**วิธีแก้:**
1. เปิด Developer Console (กด F12)
2. ดูข้อความ error
3. ตรวจสอบว่าเบอร์โทรศัพท์ถูกต้อง
4. ลองรีเฟรชหน้าเว็บ

### ปัญหา: แสกนแล้วไม่สามารถโอนได้
**วิธีแก้:**
1. ตรวจสอบว่าเบอร์ที่ใส่ผูกกับ PromptPay แล้ว
2. ลองถ่ายรูป QR Code และแสกนใหม่
3. ลองใช้แอพธนาคารอื่น

### ปัญหา: จำนวนเงินไม่ถูกต้อง
**วิธีแก้:**
1. ตรวจสอบราคาสินค้าในระบบ
2. ตรวจสอบฟังก์ชัน `getTotalPrice()`
3. ลองเคลียร์ตะกร้าและเพิ่มสินค้าใหม่
