## 🔄 สถานะคำสั่งซื้อ

```
┌─────────────────────────────────────────────────┐
│  สถานะคำสั่งซื้อ                                │
├─────────────────────────────────────────────────┤
│                                                 │
│  1. รอดำเนินการ (Pending)                       │
│     ├─> ลูกค้าสั่งซื้อและชำระเงินแล้ว          │
│     └─> รอเจ้าของร้านตรวจสอบ                    │
│                                                 │
│  2. กำลังเตรียม (Preparing)                     │
│     ├─> เจ้าของร้านยืนยันและเตรียมสินค้า        │
│     └─> กำลังจัดเตรียมสินค้า                     │
│                                                 │
│  3. จัดส่งแล้ว (Delivered)                      │
│     ├─> สินค้าถูกจัดส่งเรียบร้อย                │
│     └─> เสร็จสิ้น                                │
│                                                 │
│  4. ยกเลิก (Cancelled)                          │
│     └─> คำสั่งซื้อถูกยกเลิก                     │
│                                                 │
└─────────────────────────────────────────────────┘
```
